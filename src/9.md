# Caching

The JavaScript is cached aggressively.

Disable caching for JavaScript files:

* On browser level: Set `Disable HTTP Cache (when toolbox is open)`
  in the browsers DevTool settings
* On webserver level (development only!): `.htaccess` 
  ```js
  <FilesMatch "\.js$">
      Header set Cache-Control "no-store, no-cache, must-revalidate, max-age=0"
      Header set Pragma "no-cache"
      Header set Expires "0"
  </FilesMatch>
  ```
